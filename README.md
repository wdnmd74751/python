爬虫爬取链家网二手房3000条数据

怎么运行：lianjiasele.py是用selenium方法爬取时间比较久。先输入你的user-Agent和cookies，爬取过程中注意有时候点一下人机验证。
        lianjiacss.py是用request方法中的css爬取。爬取快但容易被封号，要爬取100页需要两个账号。
